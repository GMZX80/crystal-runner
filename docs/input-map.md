# Input Map

## Actions

| Action | Keyboard | Touch | Notes |
| --- | --- | --- | --- |
| Move left | Left arrow, A | Swipe left, Left button | Semantic lane action |
| Move right | Right arrow, D | Swipe right, Right button | Semantic lane action |
| Start/restart | Space, Enter, Start button | Start button | Same action from ready/game-over |
| Resume | Space, Enter, Resume button | Resume button | Used after visibility pause |

## Accessibility Alternatives

- Touch buttons are available for players who do not want to rely on swipe recognition.
- Keyboard controls support both arrows and A/D.
- Future improvement: remapping and reduced-motion toggle.
