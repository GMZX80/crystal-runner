# Onboarding

## First 30 Seconds

The player sees a start screen with the objective and controls, then immediately encounters crystals and hazards.

## Tutorial Strategy

No long tutorial. Teach by play:

- The path has only three lanes.
- Crystals are collectible.
- Obstacles are dangerous and red.
- Score and crystals update visibly.
- Collision displays the reason and restart button.

## Control Introduction

Keyboard and touch controls are visible before the run starts. Touch buttons stay available throughout the run.

## Retention Risks

- If early speed is too high, first failure may feel unfair.
- If touch buttons cover too much screen, mobile play may feel cramped.
- If crystal/hazard shapes are not distinct enough, accessibility suffers.
